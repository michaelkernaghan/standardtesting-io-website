FILM Token Economics & Reward Strategies

Ecosystem Overview

The FILM token ecosystem employs multiple reward mechanisms to incentivize participation, quality contributions, and long-term engagement. These mechanisms work together to create a sustainable token economy that rewards different types of participation while maintaining token value through strategic distribution.

1. Film Reward Strategy Analysis

The foundation of the ecosystem's reward structure, focusing on long-term token distribution and value creation through strategic staking and participation.

Key Features:

  • Strategic token distribution modeling
  • Long-term value analysis
  • Staking reward calculations
→ Open Film Reward Strategy Calculator

2. Creative Query Rewards

A reputation-based system that rewards quality contributions to creative queries, combining stake amounts with reputation multipliers to determine rewards.

Key Features:

  • Reputation tiers (1.0x - 1.5x multipliers)
  • Stake-weighted rewards
  • Dynamic bounty distribution
→ Open Creative Query Calculator

3. DCP Quiz Game Rewards

A daily engagement mechanism that rewards knowledge and participation through quiz completion, distributing from a fixed daily pool of 150 FILM tokens.

Key Features:

  • Performance-based rewards
  • Daily reward pool
  • Minimum participation threshold
→ Open Quiz Game Calculator

Economic Integration

These three mechanisms work together to create a balanced token economy:

  • Long-term Value: The reward strategy analysis ensures sustainable token distribution and value preservation.
  • Quality Contributions: Creative query rewards incentivize high-quality participation and expertise building through reputation mechanics.
  • Daily Engagement: The quiz system maintains regular participation and knowledge building with smaller, more frequent rewards.

Together, these systems create multiple engagement paths while maintaining token scarcity and encouraging both short-term participation and long-term commitment to the ecosystem.

Reputation System

Reputation in the DCP ecosystem functions as a non-transferable metric of user engagement and expertise, similar to a soulbound NFT. Unlike $FILM tokens, Reputation cannot be bought, sold, or transferred between users, making it a pure measure of individual contribution and participation.

Reputation Building Activities:

  • Review Quality:
    • Reviews aligning with experienced reviewer consensus can earn up to 100 reputation points
    • Significantly deviating reviews can lose up to 10 reputation points
    • System uses top 10% of reviews as quality benchmark
  • Review Timing:
    • Early reviewers receive maximum timing score (1.0)
    • Later reviews receive diminishing scores (down to 0.001)

Review Weight Components:

  • Stake Weight (40%): Based on staked amount relative to other reviewers
  • Reputation Weight (40%): Based on accumulated reputation compared to other reviewers
  • Timing Weight (10%): Based on how early the review is submitted
  • Holdings Weight (5%): Based on token holdings relative to total supply
  • Stake Ratio Weight (5%): Based on stake amount relative to personal holdings

This system encourages thoughtful, timely participation while rewarding both expertise (reputation) and commitment (stake). Growth is proportional to how well assessments align with the median evaluation, promoting quality contributions to the ecosystem.
